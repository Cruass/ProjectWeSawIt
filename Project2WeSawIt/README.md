## Cara Setup Environmet:
 Sebelum menjalankan program, pastikan komputer telah memiliki:

- GCC Compiler (MinGW)
- Terminal / Command Prompt
- Git (opsional untuk clone repository)

## Instalasi

1. Buka terminal (Command Prompt / Git Bash).

2. Clone repository dari GitHub dengan perintah berikut:

git clone https://github.com/Cruass/ProjectWeSawIt.git

3. Masuk ke folder project:

cd ProjectWeSawIt

4. Masuk ke folder program:

cd Project2WeSawIt
